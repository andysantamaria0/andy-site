Evolution
What Changed
From V1’s cool restraint to something with blood in its veins
Grand Tour V1 was beautiful but too cool — it whispered when sometimes you need the red dress to walk into the room. This iteration brings Cinnabar — a warm, authoritative red pulled from the title lettering of Fitzgerald’s Tender Is the Night first edition. The same red that draws your eye across a Technicolor frame of To Catch a Thief. It’s not a primary action color — Royal Blue holds that role. Cinnabar is the pulse. The notification dot. The live indicator. The moment your eye lands and stays.
New References

To Catch a Thief (1955)

The Italian poster’s saturated Riviera blue, the noir American poster’s moonlit silhouettes. Cary Grant and Grace Kelly on the corniche. The specific blue of VistaVision Technicolor — not navy, not cerulean, something in between. Saturated but aged, like memory.

Tender Is the Night (1934)

The first edition’s bold cinnabar red title on dark olive. The Mediterranean rendered in flat, almost gouache-like blocks of color — terracotta roofs, purple-indigo trees, golden sky. A world so beautiful it’s almost painful.

The Noir Ingredient

What these references share: beauty with an edge. The Riviera as a place where extraordinary things happen to complicated people. Romance that’s never simple. Every surface conceals a story. The design system should feel the same way — gorgeous first, then you notice the depth.

Motion
Cinematic Timing
Slow dissolves, not bouncy springs — every transition should feel like a film cut
The motion language draws from cinema: slow, deliberate reveals rather than playful bounces. Elements enter from below with a gentle fade — like a title card appearing on screen. The primary easing curve is cubic-bezier(.2, 0, 0, 1) — fast entry, slow settle. Durations live between 200–600ms.

Philosophy
Design Principles
The rules behind the rules

Cinematic Restraint

Every element placed with a director’s intention. If it doesn’t serve the scene, it doesn’t make the cut. Luxury is communicated through spacing and absence, never through decoration.

The Red Thread

Cinnabar is used like the red dress in a noir film — sparingly, deliberately, and it always means something. A notification dot, a live flight, a departure. When you see red, pay attention.

Night-First

The dark canvas isn’t an afterthought — it’s the primary stage. Nuit creates drama and makes Royal Blue and Champagne luminous. Light mode should feel like stepping from a darkened theater into afternoon light.

Literary Voice

Micro-copy reads like prose. Empty states are wistful. Error messages are gracious. Loading states suggest anticipation. “The itinerary awaits its first chapter” beats “No events found.”

Slow Dissolves

Motion references cinema: elements enter with fade-up reveals, modals dissolve in, toasts slide rather than bounce. The easing curve settles like a camera coming to rest on a composition.

The Technicolor Test

Colors should feel like they were graded for 1955 Technicolor — rich and saturated but never garish. Slightly warm, slightly aged, as if viewed through the memory of a perfect trip.

Roadmap
What a Complete System Needs
Everything from favicon to page transitions — the full inventory
Once we lock the direction, here’s what it takes to ship a complete design system that touches every pixel of Vialoure. This is roughly ordered by dependency — foundations first, then components, then polish.
Phase 1 — Foundations
Design Tokens (CSS Variables)
Complete semantic color mapping (--color-primary, --color-surface, --color-error, etc.). Light + dark mode values. Exported as CSS custom properties and JS constants.
Spacing & Grid Scale
4px base unit. Scale: 4, 8, 12, 16, 20, 24, 32, 40, 48, 64, 80. Responsive grid: 12-column with 24px gutters, max-width 800px (matching current app).
Elevation & Shadow System
3 levels: subtle (cards), medium (dropdowns, popovers), dramatic (modals). All shadows use cool blue tint on dark, warm on light. No pure black shadows.
Typography System
Complete scale with line-heights, letter-spacing, font-weight for each size. Responsive adjustments at breakpoints. Font-loading strategy (display=swap).
Border Radius Scale
4px (badges), 6px (small elements), 8px (buttons, inputs), 10px (cards), 12px (large cards), 14px (sections), 50% (avatars, pills).
Motion Tokens
Named duration/easing pairs. --motion-micro, --motion-standard, --motion-dramatic, --motion-ambient. Single easing curve for consistency.
Phase 2 — Component Library
Buttons
Primary, secondary, ghost, destructive, icon-only, loading state (spinner replaces text). Disabled states. All sizes (sm, md, lg).
Form Elements
Text input, textarea, date picker, select/dropdown, toggle, checkbox, radio. All with: default, focus, error, success, disabled states. Labels and helper text.
Cards
Trip card, event card, logistics card, member card, overview stat card, inbox email card. Hover/press states. Expanded variants for month grid and week view.
Navigation
App header (logo, user avatar, sign out). Trip sub-navigation tabs (Calendar, Members, Expenses, Inbox). View toggle (List/Week/Month). Back link. Breadcrumbs.
Modal & Dialog
Overlay + modal card. Event form modal, confirmation dialogs, destructive action confirmation (with Cinnabar accents). Enter/exit animations. Backdrop blur.
Badges & Pills
Owner/member role badges. Arrival/departure badges. Notification dots (with pulse animation). Count badges. Attendee chips (active/inactive). Cost pills.
Avatar System
Image avatars (4 sizes), color placeholder with initial, stacked avatar group with overlap, overflow counter (+N). Presence dot overlay.
Feedback & Status
Toast notifications (success, error, info). Loading skeletons for every card type. Spinner. Empty states (with literary micro-copy). Error pages.
Calendar Components
Month grid cells (trip/non-trip, today, expanded). Week columns. Day list rows. Navigation (month/week arrows, title). “Add event” button.
Happening Now Widget
Floating pill (collapsed/expanded). Inline section variant. Flight progress bar with plane icon. Live pulse dot. Stagger animation for items.
Expense & Split UI
Cost badge on event cards. Split method pills (equal, custom, proportional). Custom split inputs with running total. Currency display formatting.
Timeline & Progress
Stay timeline (bar chart with member rows). Flight progress bar. Member stay date visualization. Today marker line.
Phase 3 — Animation & Polish
Page Transitions
Fade-up on route change. List items stagger on mount. Calendar view transitions (slide between months/weeks). Modal backdrop fade.
Micro-interactions
Button hover lift + glow. Card press scale(.98). Toggle slide. Checkbox check animation. Input focus border transition. Toast slide-in/auto-dismiss.
Ambient Animations
Cinnabar pulse for live indicators. Happening Now pill glow. Skeleton loading shimmer. The notch section-divider draw animation on scroll.
Scroll-Triggered Reveals
Sections fade-up as you scroll into them. The notch divider draws itself. Staggered entry for list items entering viewport. Parallax on trip cover photo.
Phase 4 — Metadata & Brand Assets
Favicon & App Icons
Favicon (16, 32, 48px). Apple Touch Icon (180px). Android Chrome icons (192, 512px). SVG favicon for dark/light mode adaptation. The notch V mark as the icon.
Open Graph & Social
og:image template (1200×630) with Nuit background, Fraunces title, notch mark. Trip-specific og:image that includes trip name and dates. Twitter card meta.
PWA Manifest
manifest.json with theme_color (#0A1628), background_color, icons, display: standalone. iOS splash screens. Status bar styling (dark-content).
HTML Meta Tags
theme-color (#0A1628). apple-mobile-web-app-status-bar-style. Description with literary voice. Viewport with viewport-fit=cover for notched devices.
Email Templates
Trip invitation email. Event notification. Expense split notification. All styled in the Grand Tour palette with Fraunces headings and the notch mark.
Light Mode
Complete token inversion: Pearl becomes the canvas, Nuit becomes text. Surfaces shift to white with subtle warm tint. Royal and Cinnabar adjust for contrast on light backgrounds.
The Answer
Yes — every single thing.
Once the direction is locked, I can implement the complete design system directly in code. Every CSS variable, every component style, every animation, every metadata tag. The existing v- prefixed CSS gets evolved into the Grand Tour palette. Buttons gain cinematic hover states. Cards shift to Nuit and Croisette surfaces. The Cinnabar pulse appears on live indicators. Fraunces replaces Playfair Display for headings. Crimson Pro brings the literary body voice. Even the loading skeletons and empty states get the treatment. It’s a comprehensive reskin + enhancement of every visual surface in the app.
